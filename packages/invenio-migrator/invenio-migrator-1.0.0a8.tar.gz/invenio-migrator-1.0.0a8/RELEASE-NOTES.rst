===========================
 Invenio-Migrator v1.0.0a8
===========================

Invenio-Migrator v1.0.0a8 was released on September 1, 2016.

About
-----

Utilities for migrating past Invenio versions to Invenio 3.0.

*This is an experimental developer preview release.*

What's new
----------

- Initial public release.

Installation
------------

   $ pip install invenio-migrator==1.0.0a8

Documentation
-------------

   http://pythonhosted.org/invenio-migrator/
